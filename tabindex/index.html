An illustration of tabindex

Press tab to begin

First press: Focus starts here because div has tabindex='1'
Second press: Focus goes here because div has tabindex='1' and DOM order is followed.
Last press: Focus goes here because div has tabindex='0' and tabindex="0" "means that the element should be focusable in sequential keyboard navigation, after any positive tabindex values. The focus navigation order of these elements is defined by their order in the document source." (MDN)
Third press: Focus goes here because div has tabindex='2', which was added by script.
Fourth press: Focus goes here because div has tabindex='3', which was added by script.
